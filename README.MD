# 🐍 Belajar Python 2026

Repositori ini berisi kumpulan materi belajar bahasa pemrograman Python, disusun dari dasar hingga topik lanjutan. Cocok untuk pemula yang ingin memahami konsep-konsep fundamental Python secara bertahap.

## 📚 Daftar Materi

| No | File | Topik |
|----|------|-------|
| 1 | `app.py` | Hello World — pengenalan fungsi `print()` |
| 2 | `data_types.py` | Tipe data dasar (string, integer, float, boolean) dan konversi tipe |
| 3 | `expression_and_variable.py` | Variabel, operasi aritmatika, dan concatenation string |
| 4 | `control_flow.py` | Percabangan dengan `if`, `elif`, `else` |
| 5 | `loops.py` | Perulangan `for` dan `while`, penggunaan `range()` dan `break` |
| 6 | `data_structure.py` | Struktur data: list, tuple, set, dan dictionary |
| 7 | `array.py` | Metode-metode list (`append`, `pop`, `sort`, `reverse`, dll.) |
| 8 | `string_operations.py` | Operasi string: slicing, `upper()`, `lower()`, `split()`, `join()` |
| 9 | `string_operations2.py` | Latihan praktis operasi string |
| 10 | `functions.py` | Membuat fungsi dengan parameter dan return value |

## 🛠️ Cara Menjalankan

1. Pastikan Python 3 sudah terinstall:
   ```bash
   python --version
   ```
2. Install dependensi (untuk materi NumPy):
   ```bash
   pip install numpy
   ```
3. Jalankan file materi:
   ```bash
   python nama_file.py
   ```

## 📖 Topik yang Dipelajari

- **Dasar Python** — variabel, tipe data, ekspresi, dan output
- **Kontrol Alur** — percabangan dan perulangan
- **Struktur Data** — list, tuple, set, dictionary
- **String** — manipulasi dan operasi string
- **Fungsi** — definisi fungsi, parameter, dan return value
- **OOP** — class, object, constructor, dan method
- **NumPy** — komputasi numerik dengan array
